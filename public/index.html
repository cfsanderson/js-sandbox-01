Javascript Basics

Beau Teaches Javascript

This project is based on freeCodeCamp's "Javascript Basics: Beau Teaches Javascript" video playlist on YouTube by Beau Carnes.

I started using this basic vanilla JS app to follow along with the videos anyway so I figured in the spirit of freeCodeCamp I'd give it a simple UI and make it public for others to benefit.

Everything is happening in the console so pop open those dev tools and check it out. If you need some help getting this project going, check out the README.md on the Github repo. Also, feel free to tweet at me with questions, comments, concerns, or complaints.
